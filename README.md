##  Technologies Used

 - [Next Js 14](https://nextjs.org/)
 - Typescript
 - [Tremor ui](https://www.tremor.so/)
 - Tailwindcss
 - [Next Auth v5](https://authjs.dev/guides/upgrade-to-v5)
 - Mongo DB
 - Mongoose
 - Bcryptjs
 - React-icons
